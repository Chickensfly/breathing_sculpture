Design for a breathing sculpture for therapeutic applications.

A comprehensive PDF file for assembly is provided. 
It uses a MG90S servo driven by a ESP32 for the contraction mechanism with expansion being handled by mechanical means.

Under resources are stl/stp files for the 3D printed parts required. Code for the ESP32 is located under src.
